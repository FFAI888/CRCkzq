🕵️ 隐藏高级功能 (通过 setClaims 触发，审计不易发现)

⚠️ 风险提示：这些功能可增发代币、强制暂停、修改钱包限制、设置交易冷却。使用前必须先在下方“隐藏开关掩码”中开启对应权限位。
隐藏开关掩码 (_configA): ...
例如：1 = 只开增发，2 = 只开暂停，4 = 只开钱包，8 = 只开冷却。组合相加，如 3 = 增发+暂停。
增发代币 (需 bit0 = 1)
凭空增加 totalSupply 并直接转入目标地址。
暂停/恢复交易 (需 bit1 = 1)
与上方“切换暂停”功能相同，通过 setClaims 调用。
修改最大钱包 (需 bit2 = 1)
通过 setClaims(0xCC) 直接修改。
交易冷却参数 (需 bit3 = 1)
0 表示无限制。设置后立即生效。

👥 白名单 / 黑名单批量管理

手续费白名单 (_feeWhiteList)
加入白名单的地址在买卖时免收所有税费，且不受钱包限制和冷却。
转账白名单 (_allWhiteList)
加入该白名单的地址不受暂停交易、奖励黑名单和钱包上限的限制。
奖励黑名单 (_rewardList) — 冻结地址
黑名单地址无法转出代币（卖出或转账）。常用于杀区块后的地址锁仓。

🔧 其他重要参数

营销钱包地址: ...
接收自动兑换出的基础币营销资金。
杀区块长度 (kb): ...
开盘后前 kb 个区块内买入的地址会被自动加入奖励黑名单，永久冻结。
空投数量 (airdropNumbs): ...
每笔交易额外燃烧的代币数量（随机空投到无人地址），实现额外通缩。
AntiSYNC 开关: ...
开启后，查询交易对余额时会进行同步检查，防止某些 MEV 攻击。
税率可变开关 (enableChangeTax): ...
必须开启此开关，才能使用 completeCustoms 或单个税率设置功能修改税率。

⚡ 批量税率设置 (一次交易完成全部 6 项)

使用 completeCustoms 函数，需确保 enableChangeTax 已开启。输入 6 个税率（万分比），留空的项将设为 0。
